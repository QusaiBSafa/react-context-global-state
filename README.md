# React Context Global State

## Description
This is a standalone React application that utilizes the Context API to manage global state for user authentication.

## What's Built
- User authentication context
- Login and logout functionality
- Simple UI to demonstrate state management

## How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/react-context-global-state.git
   cd react-context-global-state
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the application:
   ```bash
   npm run dev
   ```
4. Open your browser and go to `http://localhost:3000`

## API Docs
- **POST /api/login**: Authenticates a user.
- **POST /api/logout**: Logs out the user.

## Environment Variables
- None required.
